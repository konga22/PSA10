# Grade Preview

포켓몬 카드 그레이딩 제출 전 상태를 미리 확인하는 모바일 웹앱입니다.

## 실행

```bash
npm test
npm run serve
```

브라우저에서 `http://localhost:4173`을 엽니다.

데모 화면은 `http://localhost:4173/?demo=1`에서 확인할 수 있습니다.

## 기능

- 앞면/뒷면 사진 업로드, 촬영, 데스크톱 드래그앤드롭
- 바깥선과 안쪽선 2중 기준선 조절
- L/R, T/B 센터링 계산
- PSA, BGS, CGC, BRG 후보 등급 표시
- 사진 품질, 코너, 엣지, 표면 자동 의심 신호
- 코너, 엣지, 표면, 변조 의심 수동 체크
- 사진 위 상/하/좌/우 단일 선 핸들 조절
- 사진 중심 기준 회전, 수직 기울기, 좌우 기울기 보정
- 자동확대 모드에서 선 핸들 드래그 중 해당 부분 확대
- 바깥선과 안쪽선 사이 빗금 표시
- 다크모드 지원
- 코너, 엣지, 표면 수동 확인 가이드와 깨끗함/미세/보임/심함 기준 표시
- 등급사별 센터링 기준과 수동 체크 상태를 비교한 상세 예상 근거
- BGS 서브그레이드형 결과 표시

## 기준 문서

등급사별 기준과 구현 계획은 [docs/grading-precheck-plan.md](./docs/grading-precheck-plan.md)에 정리되어 있습니다.

## 현재 판정 원칙

이 앱은 공식 등급을 확정하지 않습니다. 사진과 수동 체크를 바탕으로 후보 등급, 감점 사유, 신뢰도를 보여주는 프리체크 도구입니다.

`미세`는 확대나 빛 각도에서만 보이는 작은 결함을 뜻합니다. 단독 미세 결함은 PSA/BRG에서는 바로 hard downgrade하지 않고 10 리스크로 표시하며, BGS는 서브그레이드 구조 때문에 해당 항목 9.5 리스크로 더 민감하게 반영합니다. 여러 항목이 미세이면 상위 등급 후보 범위를 낮춥니다.
